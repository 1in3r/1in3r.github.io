正文

更换一下exp中的 ip 即可

javac 编译一下

使用工具开启一个 RMI 和 LDAP 服务

工具地址：https://github.com/RandomRobbieBF/marshalsec-jar

nc 监听 12000 端口

Exploit

Reverse shell

JNDI注入测试工具

作者github地址：https://github.com/welk1n/JNDI-Injection-Exploit

写一个反弹shell脚本，并使用python开启一个简单http服务以供目标下载

使用JNDI-Injection-Exploit启动一个RMI服务器、LDAP服务器和HTTP服务器

让目标下载反弹shell脚本

让目标执行反弹shell脚本

docker pull registry.cn-hangzhou.aliyuncs.com/fengxuan/log4j_vuln
docker run -it -d -p 8080:8080 –name log4j_vuln_container registry.cn-hangzhou.aliyuncs.com/fengxuan/log4j_vuln
docker exec -it log4j_vuln_container /bin/bash
/bin/bash /home/apache-tomcat-8.5.45/bin/startup.sh
